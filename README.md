# PathOfSolace

**PathOfSolace** is a work-in-progress RogueLite game built in C# using .NET and MonoGame.

## Overview
This project contains the game code for a RogueLite inspired game.  
Built with:
- C# as the core language
- .NET for platform support
- MonoGame as the game framework

## Game Features
- Grid based maps and movement.
- Basic collision with environment / entities.
- Basic enemy aggro and pathfinding AI.
- Basic combat between two units.

## Planned Features
- Basic chatbox for user to read what is happening in the world.
- Basic menus for character abilities and equipment.
- Character stats for health, mana, defence, attack, etc.
- Advanced combat system taking stats into account.
- Transition between different maps states.
- Preset map stages for cities / towns / world events.
- Random world generation for maps on random tiles in the game world.
- Game world traversal screen.

## Running the Game
1. Clone the repository
2. Install Dotnet
3. Open PowerShell in game directory
4. Run `dotnet run` command into PowerShell.
